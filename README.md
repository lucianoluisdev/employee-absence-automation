# Employee Absence Automation

Automação em Python para envio automático de relatórios de faltas para gestores.

## Tecnologias utilizadas

* Python
* Pandas
* PyAutoGUI
* Pyperclip
* Excel

## Funcionalidades

* Leitura automática de planilhas Excel
* Tratamento de dados
* Cruzamento de informações
* Separação de colaboradores por setor
* Envio automático de e-mails
* Automação de processos de RH

## Estrutura do Projeto

```text
employee-absence-automation/
│
├── README.md
├── automacao_rh.py
├── requirements.txt
│
└── dados_exemplo/
    ├── Cadastro_Colaboradores.xlsx
    ├── Base_Faltas_Colaboradores.xlsx
    └── Gestores_Setor.xlsx
```

## Como executar

```bash
pip install -r requirements.txt
```

Depois execute:

```bash
python automacao_rh.py
```
